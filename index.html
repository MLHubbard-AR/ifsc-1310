Thoughts on Progressive Enhancement

Progressive Enhancement is a useful tool because it can help to build your website that can work (or be used) on many different devices. It's main focus is on the content. I like the example that was given about the M&M's. It brought things into perspective. What I got from this reading is that if we start with the content and work our way outward to the presentation and the end result, we can come up with a great website. Not only will it be a great site but with the application of JavaScript to help with the styling of the content. You can't go wrong.
